MySQL 语句 ｜ find_in_set()

FIND_IN_SET() 是 MySQL 中的一个字符串函数，用于在逗号分隔的字符串列表中查找指定字符串的位置。

它的语法如下：

  • search_string: 要搜索的字符串。
  • string_list: 逗号分隔的字符串列表。

该函数返回一个整数值，表示在字符串列表中找到搜索字符串的位置。如果未找到，返回0。
